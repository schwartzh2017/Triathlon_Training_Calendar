## Swim — 35–40min (~1,500 yards)

**Warm-up — 300 yards easy freestyle**

**Drill Set — 150 yards**

3×50 yards with 20–25s rest:
- Catch-up drill
- Single-arm drill
- Fingertip drag

**Main Set — 600 yards**

*Same as last week — hold the 200yd structure, improve consistency.*

3×200 yards @ aerobic effort, 30s rest between.

Target pace: 1:46–1:52/100yd.

Focus: stroke count at 11 or below per length.

**Speed Set — 300 yards**

*Progression from last week's 5×50yd.*

6×50 yards at race pace effort, 15s rest.

Target pace: 1:33–1:38/100yd.

*One extra 50 added — building toward race pace volume.*

**Cool-down — 150 yards easy**

*Total: ~1,500 yards*
