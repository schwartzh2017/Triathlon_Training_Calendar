## Swim — 35min (~1,500 yards)

**Warm-up — 400 yards easy freestyle**

**Drill Set — 200 yards**

4×50 yards alternating:
- Catch-up drill (high elbow catch, lat engagement)
- Fingertip drag (high elbow recovery)

20–25s rest between lengths.

**Main Set — 600 yards**

6×100 yards @ comfortable aerobic effort, 20s rest.

Target pace: 1:45–1:50/100yd.

*One extra 100 added from last week.*

**Speed Set — 200 yards**

4×50 yards slightly stronger effort, 15s rest.

Target pace: 1:35–1:40/100yd.

**Cool-down — 100 yards easy**

*Total: ~1,500 yards*

---

## Sauna

10–12min post-swim.

Continue progressive heat adaptation — target 200°F+.

## Progression notes

Bench: hold at 90 lb — consistency over weeks builds through the ceiling.

Skull crusher: if 45 lb felt RPE ≤ 8, try 50 lb top set next Arms B.

Bench fly: 17.5 lb confirmed as new working weight.

Hollow body: if 3 sets at 5/5/10 lb felt controlled, move to 5/10/10 lb next Arms B.
