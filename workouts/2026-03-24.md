**Day:** Tuesday

**Estimated time:** ~25min plyos + 45min lift

**Session order:** Plyometrics → Lift

**Context:**

Plyometrics come first — they are neuromuscular work and require a fresh nervous system.

Arms A (back/biceps) pairs well here because plyos are purely lower body — no conflict with the upper body pulling work.

45min hard cap on the lift today.

---

## Non-negotiables before you start

Collagen + tart cherry juice 60min before session start.

Shin check before plyos: tap tibia firmly.

If any point tenderness, skip single-leg bounding and box hop downs.

Do a proper 5min warm-up jog or bike before plyos — never start cold.

---

## Plyometrics Session

*~25min total including warm-up*

**Bilateral work — 30s on / 15s rest:**

| Exercise |
|---|
| Double hops in place |
| Forward / backward hops |
| Triple hops |
| Side to side hops |
| Scissor hops |
| Walk + hop (alternating double/single) |
| Box hop down + jump |

**Unilateral work — 20s per side / 20s rest between exercises:**

| Exercise |
|---|
| Single leg forward / backward |
| Single leg triple hop |
| Single leg side to side |
| Single leg lateral bounds |
| Single leg box hop downs |

*Shin check again after the single-leg box hop downs.*

*If anything feels off at any point: stop the plyo session and move straight to the lift.*
