**Context:**

Plyometrics come first — they are neuromuscular work and require a fresh nervous system.

Arms A (back/biceps) pairs well here because plyos are purely lower body — no conflict with the upper body pulling work.

45min hard cap on the lift today.

---

## Non-negotiables before you start

Collagen + tart cherry juice 60min before session start.

Shin check before plyos: tap tibia firmly.

If any point tenderness, skip single-leg bounding and box hop downs.

Do a proper 5min warm-up jog or bike before plyos — never start cold.

### 2. Superset — Band-Assisted Pull-Up + Reverse Fly

*No rest between A and B. 60s after each full superset.*

**2A. Band-Assisted Pull-Up**

*Strong band. Full hang at the bottom, chin over bar at the top.*

*This replaces the band lat pulldown — it's more swim-specific and trains the same lat/scapular pattern with greater range of motion and load.*

| Set | Reps | Notes |
|---|---|---|
| 1 | 5 | First time — find the groove, don't rush |
| 2 | 5 | Focus on initiating with the lats, not the arms |
| 3 | 5 | Slow the eccentric — 3s lowering |

*If 5 reps feels too easy on all three sets: note it and we'll reduce band thickness next Arms A.*

*If you can only get 3–4 reps cleanly: stop there, don't grind ugly reps.*

### 4. Core

*No rest between A and B. 45s after each pair.*

**4A. Pallof Press — Band (Anti-rotation)**

*Anchor band at chest height. Stand sideways to anchor point, feet shoulder-width. Press band straight out, hold 2s, return. Keep hips square — do not rotate.*

| Set | Reps | Notes |
|---|---|---|
| 1 | 10 each side | Light-moderate band tension |
| 2 | 10 each side | |

**4B. Dead Bug (Anti-extension)**

*Lie on back, arms straight to ceiling, knees bent 90° in the air. Lower opposite arm and leg toward floor simultaneously while pressing lower back into ground. Return and alternate.*

| Set | Reps |
|---|---|
| 1 | 8 each side |
| 2 | 8 each side |

*If time is short, do one set of each instead of two.*

## Progression notes

If row top set (85 lb) felt RPE ≤ 8: move to 90 lb top set next Arms A.

If curls at 22 lb felt solid: move to 22/24 lb next Arms A.

Pull-up feedback needed: report how many clean reps you got per set and whether the strong band felt like enough assistance or too much.

That data sets the progression path — band thickness reduction vs rep increase.
