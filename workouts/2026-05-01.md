## Non-negotiables before you start

Collagen + tart cherry juice 60min before starting.

Shin and achilles check before plyos.

## Plyometrics Session

*~15min*

**Bilateral — 30s on / 15s rest:**

| Exercise |
|---|
| Double hops in place |
| Front/back hops |
| Triple hops |
| Side to side hops |
| Scissor hops |
| Jumping across room, landing on alternating legs |
| Double leg bench drop + hop |

**Unilateral — 20s each side / 10s rest after both sides:**

| Exercise |
|---|
| Single leg hops in place |
| Single leg front/back hops |
| Single leg triple hops |
| Single leg side to side hops |
| Single leg lateral bounds |
| Single leg bench drop + hop (half height) |

*Shin and achilles check after single-leg bench drop.*

*Any sensation: stop plyos, skip PT, go straight to easy run or rest.*

## Run — Longer Easy Aerobic

**Duration: 40–45 minutes**

**HR ceiling: 152 bpm.**

**Effort:** Genuinely easy — this is volume, not quality.

Tuesday had the tempo intervals. Today is about accumulating aerobic time on feet.

**No strides today.**

**Pacing:** Start at 10:00–10:30/mi for the first 10 minutes regardless of how good you feel. Coming off plyos your legs will be primed but not necessarily fresh — don't let that initial energy pull you into a faster pace.

---

| Metric | Target |
|---|---|

## Achilles protocol — mandatory post-run

Straight-leg calf stretch: 30s each side × 2 rounds.

Bent-knee calf stretch: 30s each side × 2 rounds.

---

## What to report back

Duration, avg HR, achilles status.

How legs felt post-plyos — heavy, primed, or neutral going into the run.

Any shin sensation during plyos or run.
