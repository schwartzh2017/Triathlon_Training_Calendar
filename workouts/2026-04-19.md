## Context — Why Sunday is now a run day

You have three distinct run sessions per week in Race Prep:

- **Thursday:** Quality run — strides and eventually pace work
- **Saturday:** Brick run — transition adaptation, short by design
- **Sunday:** Longer aerobic run — building the run base you need for race day

The brick run on Saturday is intentionally short (15–20 min) because its purpose is transition adaptation, not fitness.

This Sunday session is where actual run fitness gets built.

Target: build from 35 min today toward 50–55 min by race day on May 31.

---

## Non-negotiables before you start

Collagen + tart cherry juice 60min before heading out.

Shin check before heading out.

Achilles check — note status.

You are coming off Saturday's brick — legs will carry some fatigue.

Start slower than feels necessary.

---

## The Run

**Duration:** 35–40 minutes continuous.

**HR ceiling: 152 bpm.**

**Effort:** Genuinely easy aerobic — same controlled effort as your best recent runs.

This is not a fitness test after yesterday's brick.

It is a deliberate aerobic stimulus on mildly fatigued legs — which is exactly what race day demands in the final kilometers of the run.

**Pacing:** Start at 10:00–10:30/mi for the first 10 minutes regardless of how good you feel.

Let HR settle before finding your natural rhythm.

No strides today — Saturday had the intensity work.

Today is volume.

## Optional sauna

If available after the run: 15–20min at 180–200°F.

Contributes toward weekly heat exposure target.

Good parasympathetic recovery after the weekend's training load.
