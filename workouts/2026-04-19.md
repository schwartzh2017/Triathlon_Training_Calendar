## Context — Three run sessions per week

- **Thursday:** Quality run — strides and eventually pace work
- **Saturday:** Brick run — transition adaptation, short by design

---

## Non-negotiables before you start

Collagen + tart cherry juice 60min before heading out.

Shin check before heading out.

Achilles check — note status.

You are coming off Saturday's brick — legs will carry some fatigue.

Start slower than feels necessary.

---

## The Run

**Duration:** 35–40 minutes continuous.

**HR ceiling: 152 bpm.**

**Effort:** Genuinely easy aerobic.

This is not a fitness test after yesterday's brick.

**Pacing:** Start at 10:00–10:30/mi for the first 10 minutes regardless of how good you feel.

Let HR settle before finding your natural rhythm.

No strides today — Thursday and Saturday had the intensity work.

Today is volume.

### 2. Superset — Band-Assisted Pull-Up + Face Pull

*No rest between A and B. 60s after each full superset.*

**2A. Band-Assisted Pull-Up — Neutral Grip**

*Trial the medium band this week if the strong band felt easy last session.*

*If medium band is too hard and reps drop below 4: go back to strong band.*

| Set | Reps | Notes |
|---|---|---|
| 1 | 5 | Medium band — assess immediately |
| 2 | 5 | 3s eccentric — slow and controlled |
| 3 | 5 | Note honest difficulty vs strong band |

**2B. Face Pull — Band (replaces reverse fly)**

*Anchor band at face height.*

*Stand facing anchor, elbows high and wide, pull toward your face with external rotation — finish with hands beside ears, thumbs pointing back.*

*This is the most swim-specific shoulder exercise in your program — directly trains the muscle activation pattern of the catch and pull phase.*

| Set | Reps | Notes |
|---|---|---|
| 1 | 12 | Light-moderate band — establish feel |
| 2 | 12 | Focus on external rotation at end range |
| 3 | 12 | Elbows must stay high throughout |

*New movement — reps higher than usual (12) because band tension is harder to quantify. Focus on quality of external rotation.*

## Progression notes

Row: if 95 lb top set felt RPE ≤ 8, move to 95 lb working set next Arms A.

Pull-ups: report whether medium band was manageable or too hard.

Face pull: note how the external rotation felt — any shoulder tightness or weakness.

Bicep curl: if 22.5 lb top set held 5 reps, try 25 lb top set next Arms A.

Copenhagen plank: if 20s felt controlled, move to 25s next Arms A.

## What to report back

Run: duration, avg HR, max HR, achilles status, how legs felt post-brick.

Pull-up: medium band assessment.

Face pull: how the movement felt — new so baseline needed.
