## Progression notes

Trap bar: 200–205 lb top set this week establishes the Race Prep progression ceiling.

Squat: 170 lb is a meaningful jump — if it feels RPE 9+, hold at 165 lb and build density.

Hip thrust bilateral: 235 lb top set — first push beyond 225 lb.

Single-leg hip thrust: note left/right rep difference — watching for improvement.

Lateral lunge: moved to bilateral dumbbell hold — start conservative.
